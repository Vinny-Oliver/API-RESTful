Instalando uma ferramenta do Entity Framework EF
Nível de máquina e não de projeto
dotnet tool install --global dotnet-ef

Instalando o Pacote Entity Framework - Instalado no projeto
Criou um EF deverá instalar sempre ele
dotnet add package Microsoft.EntityFrameworkCore.Design

Todo pacote deverá ser instalado em cada novo projeto
dotnet add package Microsoft.EntityFrameworkCore.SqlServer
